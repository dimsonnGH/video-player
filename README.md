## Учебная верстка видеоплеера.  

Верстка выполнена на основе [библиотеки видеоплеера](https://github.com/devmanorg/video-player-jslib).

В проекте были использованы шрифты [Font Awesome](https://fontawesome.ru/)

Внешний вид видеоплеера:
![внешний вид видеоплеера](https://github.com/dimsonnGH/video-player/blob/master/static/example.png)

Ознакомиться с работой видеоплеера можно на хостинге GitHub Pages по адресу:
[https://dimsonngh.github.io/video-player/](https://dimsonngh.github.io/video-player/)

### Цель проекта

Проект создан в образовательных целях в качестве задания на курсе по верстке на сайте [dvmn.org](https://dvmn.org/).
